1. О программе
Плагин внутреннего просмотра uLister поддерживает более 500 различных форматов файлов.

2. Поддерживаемые форматы
Множество популярных форматов текстовых процессоров, электронных таблиц, презентаций, растровых и векторных изображений.
Полный список можно посмотреть в файле OutsideIn_8.5.7_Supported_File_Formats.pdf

5. Настройка
Параметры хранятся в файле ulister.ini, поиск которого производится в следующем порядке:
1) каталог плагина 
2) каталог, в котором находится %COMMANDER_INI% 
3) %APPDATA% 
Файл не будет загружен, если длина пути превысит 260 символов (ограничение Windows).

Имя секции настроек - [ulister].

В версии 4.0.0.4 плагина доступны следующие параметры:

keepinmemory        (по умолчанию 1) - не выгружать библиотеки Outside In Viewer из памяти после первого использования, что уменьшает
                    время загрузки документа, но увеличивает потребляемую память.
noloadtypes         список запрещенных к загрузке форматов
onlyloadtypes       список разрешенных к загрузке форматов
nopreviewtypes      список запрещенных к показу эскизов форматов
onlypreviewtypes    список разрешенных к показу эскизов форматов

Четырехзначные коды форматов с описанием находятся в файле formats.txt.
В качестве разделителя можно использовать любой символ, отличный от цифр, например, запятую или пробел.
Если задана разрешающая строка, то аналогичная запрещающая строка игнорируется.

optionsdir          путь к каталогу, в котором создается подкаталог ".oit" с внутренними настройками библиотеки Oracle Outside In Technology: Viewer Technology;
                    поддерживаются %переменные% окружения;
                    ограничение Windows - длина пути не более 260 символов;
                    для коректной работы optionsdir необходимо использовать пропатченную версию файла sccut.dll,
                    некоторые исправленные версии этого файла лежат в каталоге "OIT_DATA_PATH support" (например, последние, поддерживающие Windows XP)

Если параметр optionsdir не указан, то библиотека создает ".oit" в %APPDATA%, если это значение не определено, то в %LOCALAPPDATA%,
если это значение тоже не определено, то в каталоге с библиотекой.

6. Установка
Библиотеку Oracle Outside In Technology: Viewer Technology можно загрузить на странице
https://www.oracle.com/middleware/technologies/outside-in-technology-downloads.html

Установите плагин uLister.

Загрузите oit-8.5.8.204-vw-win-x64.zip для 64-разрядной версии TC и oit-8.5.7.243-vw-win-x32.zip для 32-разрядной версии. Распакуйте файлы.
(Если нужна поддержка Windows XP x64 SP2 или Windows XP x86 SP3, то используйте более старые версии oit-8.5.5.12-vw-win-x86-64.zip и oit-8.5.5.12-vw-win-x86-32.zip)

Если вы распаковали 64-битную библиотеку, то 64-разрядные файлы из распакованного каталога redist скопируйте в каталог redist64 плагина.
Если вы распаковали 32-битную библиотеку, то 32-разрядные файлы из распакованного каталога redist скопируйте в каталог redist32 плагина.

Если вы планируете использовать параметр optionsdir плагина, то замените файл sccut.dll соответствующей исправленной версией из каталога "OIT_DATA_PATH support".

7. Сочетания клавиш
Перед "поиском-далее" по F3 всегда явно начинайте с вызова диалога поиска (F7 или Ctrl-F).

8. Компиляция.
Используйте файл vs2015.sln для редактирования исходных кодов проекта в современных версиях Visual Studio.
Используйте файл vs2005.sln для компиляции проекта в компактный код с поддержкой Windows XP 32/64 с помощью
Visual Studio 2005 (для компиляции в x64 понадобится версия Pro).
Затем используйте скрипт MAKECAB.CMD для создания файла автоматической установки плагина wlx-ulister.cab для Total Commander.
